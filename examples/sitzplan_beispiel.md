# Beispiel — Sitzplan (Gemeinschaftsschule Sachsen-Anhalt)

> **Beispielcharakter.** Vorlage für einen Sitzplan mit Kürzeln (statt echter Namen). Vor Verwendung sind die schulischen Datenschutzregeln zu beachten. Keine Diagnosen, keine schulbezogenen personenbezogenen Daten.

## Pflicht-Kopf

```
Schulform:       Gemeinschaftsschule Sachsen-Anhalt
Bildungsgang:    [Bildungsgang — Ebene 3 Annahme]
Lehrplanbasis:   [Sekundarschule / Gymnasium — Ebene 3 Annahme]
Fach:            [Deutsch / Evangelische Religion — Ebene 3 Annahme]
Klasse:          [Klasse — Ebene 3 Annahme]
Raum:            Klassenraum 1.07 (Beispiel)
Anzahl SuS:      24 (Beispiel)
Raumform:        U-Form (Variante A — siehe unten)
```

> **Wichtig:** Alle Personenkennungen sind **Kürzel** (S1 … S24). Keine echten Schüler*innennamen. Keine Diagnosen, keine medizinischen/psychologischen/sonderpädagogischen Klassifizierungen.

## Pädagogische Begründung (Ebene 2)

- Ruhebedürfnis und gute Sicht auf Tafel/Beamer berücksichtigt.
- „Neutrale Zonen" für konzentriertes Arbeiten.
- Sitznachbar*innen so gewählt, dass Störungsanfälligkeit minimiert und Mitarbeit gefördert wird.
- Konflikte / Trennwünsche werden funktional und ohne Diagnose beschrieben (z. B. „S7 und S12 sollten nicht direkt nebeneinander sitzen — bekannte Konfliktdynamik aus dem vorherigen Halbjahr").
- Freundschaftscluster (S16 + S17, S20 + S21) erhalten Blickkontakt, aber keine direkte Nachbarschaft (geringere Ablenkungsgefahr).

## Hinweise zur Sitzplatzwahl (Ebene 2)

- **Förderbedarf** wird **funktional** beschrieben (z. B. „braucht kurze Wiederholung", „förderlich in ruhiger Umgebung") — keine Diagnosen.
- **Konflikte** werden neutral benannt (z. B. „nicht nebeneinander platzieren").
- **Freundschaften** werden berücksichtigt, aber nicht zwingend direkt nebeneinander gesetzt (Ablenkungsgefahr).
- **Trennwünsche** der SuS / Eltern werden respektiert, soweit pädagogisch vertretbar.
- **Sicht auf Tafel/Beamer** und **Hörweite** sind harte Kriterien.
