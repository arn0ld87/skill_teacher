# Checkliste: Teacher-Skill Gemeinschaftsschule Sachsen-Anhalt

## Abdeckung

- [ ] Alle Klassen 5 bis 12 berücksichtigt
- [ ] Deutsch abgebildet
- [ ] Evangelische Religion abgebildet
- [ ] Kein katholischer RU enthalten
- [ ] Keine Ethik enthalten
- [ ] Keine Klassen außerhalb 5–12 (z. B. 13) enthalten

## Lehrplan-Struktur

- [ ] Sekundarschul-Lehrpläne getrennt gespeichert
- [ ] Gymnasium-Lehrpläne getrennt gespeichert
- [ ] Gemeinschaftsschul-Routing definiert
- [ ] Klasse 5–8 nutzt primär Sekundarschule
- [ ] Klasse 9–12 prüft Bildungsgang
- [ ] Klasse 9–10 unterscheidet Sekundarschulabschluss/mittleren Abschluss vs. gymnasialen Bildungsgang
- [ ] Klasse 10 gymnasial korrekt als Einführungsphase geroutet
- [ ] Klasse 11–12 ausschließlich Gymnasium (Qualifikationsphase)
- [ ] Abiturbezug nutzt Gymnasium-Fachlehrpläne
- [ ] Unklarer Bildungsgang führt zu Rückfrage oder sichtbar markierter Annahme

## Inhalt & Qualität

- [ ] Kompetenzmatrix vorhanden
- [ ] Quellenbezug vorhanden (Datei/Basis/Fach/Klasse/Abschnitt/Seite)
- [ ] Datenschutzregeln enthalten
- [ ] Workflows vorhanden
- [ ] Templates vorhanden
- [ ] Beispiele vorhanden

## Datenschutz & Verlässlichkeit

- [ ] Keine echten Schülerdaten
- [ ] Keine echten Schülernamen
- [ ] Keine echten Diagnosen
- [ ] Keine unbelegten Lehrplanbehauptungen
- [ ] Vier-Ebenen-Kennzeichnung vorhanden (1 Lehrplan / 2 didaktisch / 3 Annahme / 4 zu prüfen)
- [ ] Fehlerverhalten definiert (z. B. fehlende Quelle, fehlender Bildungsgang, Klasse außerhalb Bereich)
