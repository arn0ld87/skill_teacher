# Prüfkriterien für examples/

Jede Datei in examples/ muss die folgenden sichtbaren Angaben enthalten:

- [ ] Lehrplanbasis (Sekundarschule oder Gymnasium) explizit genannt
- [ ] Bildungsgang explizit genannt (bzw. Rückfrage/Annahme dokumentiert, falls unklar)
- [ ] Lehrplanbezug mit Quelle (Datei/Basis/Fach/Klasse/Abschnitt/Seite)
- [ ] Getroffene Annahmen sichtbar als Annahme markiert (Ebene 3)
- [ ] Eigentliche Ausgabe (Unterrichtsmaterial/Klassenarbeit/Klausur etc.) klar vom Metadaten-Block getrennt
- [ ] Qualitätscheck/Selbstprüfung am Ende sichtbar (z. B. Kompetenzabgleich, Vollständigkeit)

Zusätzliche Prüfungen je Beispiel:

- [ ] Keine echten Schülernamen enthalten
- [ ] Keine echten Diagnosen oder personenbezogenen Daten enthalten
- [ ] Vier-Ebenen-Kennzeichnung konsistent verwendet (1 Lehrplan / 2 didaktisch / 3 Annahme / 4 zu prüfen)
- [ ] Klasse liegt im Bereich 5–12
- [ ] Fach ist Deutsch oder Evangelische Religion (kein kath. RU, keine Ethik)
- [ ] Bei Klasse 9–10: Bildungsgangzuordnung nachvollziehbar dokumentiert
- [ ] Bei Klasse 11–12: ausschließlich Gymnasium referenziert, kein Sekundarschulbezug
- [ ] Bei Abiturbezug: Kennzeichnung "Abiturbezug" vorhanden und Quelle stammt aus Gymnasium-Fachlehrplan
- [ ] Keine unbelegte Lehrplanbehauptung ohne Quellenangabe oder Ebene-4-Markierung
